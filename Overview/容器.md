 # 1. <span style="color:red;">向量 ```vector```
   ```#include <vector>  ``` 

    数组一样类型的存储结构，但长度可变

  * ## 1.1.基本构造
      ```c++
    vector<int> arr1;               //一个int类型的空数组
    vector<int> arr2(100)           //一个int类型的100个元素的数组
    vector<int> arr3(100,1);        //一个int类型的100个元素的数组，初值为1   
    vector<vector<int>> mat(100)    //初始100行的二维数组，未指定列数，不能直接用，会超出界限
    vector<vector<int>> mat(100,vector<10，1>)   //初始100行10列的二维数组，初值为1
    vector<vector<vector<int>>> dp(5,vector<vector<int>> (4,vector<int> (3,1)))    //一个等价于dp[5][4][3]的三维数组，初值为1
    ```
    对于未指定的行数列数的vector不能进行访问操作
  
  * ## 1.2.部分基本用法

    *```.push_back(元素)``` , ```.pop_back()```进行增删操作*

    *```.size()```获取长度*

    *```.clear()```清空vector，```.empty()```判断vector是否为空*

    *```.resize()```直接更改长度，若为扩大，则新元素均为默认值（原有的不变）*

    *```.insert(迭代器，值)```在迭代器指向的位置前插入元素*

    *```.erase(迭代器)/erase(起始迭代器,结束迭代器)```删除迭代器指向的元素*

      * *迭代器：类似下标，包括 ```.begin()``` ```.end()```和遍历出的 ```auto it```* 

    ```c++
    vector<int> arr;
    arr.push_back(1);   //arr[0]为1
    arr.push_back(2);   //arr[1]为2
    arr.push_back(3);   //arr[2]为3
    arr.pop_back();     //arr[2]在此时被删除
    arr.size();         //返回arr的长度，此时为2
    arr.clear();        //清空arr
    arr.empty();        //若arr为空返回1(true)，不为空则返回0(false)，此时为1
    arr.resize(3,10);   //更改arr长度为3，且元素均为默认值10
    arr.erase(arr.begin()+1); //删除了"1"位(第二位)
    arr.insert(arr.end()-1);  //删除了最后一位
    ```
  * ## 1.3.注意事项
 
      ### 1.3.1）提前指定长度
      尽量在构造函数时指定长度，而不是一个一个 ```.push_back()``` ，这样会浪费很多时间开销
      ### 1.3.2）注意 size_t 溢出
      使用```.size()```获取长度的返回值为 size_t 类型，在OJ平台中一般的32位系统，即size_t的取值范围是$[0,2^{32})$

      * *注意区别 size_t 和 int 类型的区别，size_t为$[0,2^{32})$，int为$(-2^{31},2^{31})$*

 # 2. <span style="color:red;">栈 ```stack```
   ```#include <stack>```

   通过二次封装双端队列容器，实现 **先进后出** 的栈数据结构

  * ## 2.1构造
    ```c++
    stack<int> stk; //一个 int 类型的栈 stk
    ```
  * ## 2.2基本用法
    |作用|用法|示例|
    |:-:|:-:|:-|
    |进栈|```.push(元素)```|``` stk.push(1) //加入1到栈顶```|
    |出栈|```.pop()```|```stk.pop()    //将栈顶的元素删除```|
    |取栈顶|```.top()```|```stk.top()  //查看栈顶元素```|
    |查长度|```.size()```|```cout<<stk.size();  //返回值为这个栈的长度```|
    |判空|```.empty()```|```if(stk.empty())  //若为空返回1，否则为0```|
  * ## 2.3注意事项
    ### 2.3.1） vector 可模拟栈的用法
      使用 ```vector.back``` 相当于取栈顶， ```.push_back()``` 相当于进栈， ```.pop_back()``` 相当于出栈， stack 默认由 vector 实现
    ### 2.3.2） **不可访问内部元素**
      stack只允许访问```stk.top()```栈顶元素，遵循先进后出，无法访问中间和前面的元素
    ### 2.3.3） stack 自身没有```.clear()```用法（后面的quene也一样）
      要一次性清除所有元素，要用```stack<int>().swap(stk)```

 # 4. <span style="color:red;">优先队列 ```priority quene```（堆）
 ```include <quene>```

 提供 **常数** 时间的最大元素查找， **对数** 时间的插入与提取，底层原理是 **二叉堆**
  * ## 4.1构造
    ```c++
    priority_quene<类型，容器，比较器>pque;
    ```
    * 类型：要储存的数据类型
    * 容器：储存数据的底层容器，默认为```vector<类型>```，竞赛一般保持默认
    * 比较器：比较大小使用的比较器，默认为```less<类型>```，可自定义
    ```c++
    priority<int> pque1;                          //储存int的大顶堆（堆顶为最大数）
    priority<int,vector<int>,greater<int>>pque2   //储存int的小顶堆（堆顶为最小数）
    ```
  * ## 4.2基本用法
    |作用|用法|示例|
    |:-:|:-:|:-|
    |进堆|```.push(元素)```|```pque.push(1) //加入1到队尾```|
    |出堆|```.pop()```|```pque.pop()    //将队首的元素删除```|
    |取堆顶|```.top()```|```pque.top()  //查看队首元素```|
    |~~判空，取长度~~|~~同上~~|~~同上~~|
    ---
    *进出堆复杂度$O(\log{n})$，取堆顶复杂度$O(n)$*

    * 每次向堆中插入大小不定的元素，或去除最大最小的元素，元素数量n，插入操作数k
      * 每次插入后进行快排：$k*n*\log{n}$
      * 使用优先队列维护：$k*\log{n}$

  * ## 4.3注意事项
    ### 4.3.1）仅堆顶可读
      仅可访问堆顶，其他元素都不可读到

      ~~```cout << pque[1] << endl ;```~~```//错误示范```

    ### 4.3.2）所有元素不可修改（包括堆顶）
      与stack可直接修改栈顶```.top()```，quene可直接修改队首```.front()```尾```.back()```不同

      priority quene只能通过```.push()```和```.pop```间接修改堆顶元素 
      
      *（新加入的元素加完后不一定是堆顶）*

 # 5. <span style="color:red;">集合```set```
 ```#include <set>```

 提供 **对数** 时间的插入，删除，查找的集合数据结构，底层原理是 **红黑树**

 | 集合三要素|解释|set|multiset|unordered_set|
 |:-:|:-:|:-:|:-:|:-:|
 |确定性|集合中的元素要么属于要么不属于|✔️|✔️|✔️|
 |互异性|集合中的元素不能重复|✔️|❌(任意次)|✔️|
 |无序性|集合中的元素没有顺序|❌(从小到大)|❌(从小到大)|✔️|

  * ## 5.1 构造
    ```c++
    set< 类型 , 比较器 > st;
    ```

  * ## 5.2 基本用法
    |作用|用法|示例|
    |:-:|:-:|:-|
    |插入|```.insert(元素)```|```st.insert(1) //插入1```|
    |删除|```.erase(元素)```|```st.erase(1) //删除1```|
    |查找|```.find(元素)```|```st.find(1) //返回1的迭代器(若无则返回st.end())```|
    |查找个数|```.count(元素)```|```st.count(1) //返回1的个数（在集合中为0或1）```|
    |判空，清空，取长度|```.empty()``` ```.clear()``` ```.size()```|同 ```vector```|

    **增删查**时间复杂度 $O(\log{n})$

  * ## 5.3 遍历
    可使用 **迭代器** 进行遍历
    ```c++
    for( set<int>::iterator it = st.begin() ; it != st.end() ; ++it)
      cout<< *it <<endl; //迭代器it是一个指针
    ```

    基于范围的循环（C++11）
    ```C++
    for(auto &变量名 : st)
    cout<< 变量名 <<endl; //此处变量是st的元素
    ```

  * ## 5.4 适用情形
    * 元素去重: $[1,1,3,2,4,4]$ -> $[1,2,3,4]$
    * 维护顺序: $[1,5,3,7,9]$ -> $[1,3,5,7,9]$
    * 元素是否出现过

  * ## 5.5 注意事项
    **1. 不能使用下标索引访问**
    
    ~~```cout<< st[1] ;   //错误```~~

    **2. 元素只读**

    set迭代器取到的元素是只读的，不可直接修改，需要先```erase```再```insert```

    ~~```*st.begin() = 1; //错误```~~   

    **3. 不可通过迭代器计算下标差值**

    set的迭代器不能像vector一样计算下标差值

    ~~```int idx = it - st.begin() //错误```~~

 # 6. <span style="color:red;"> 映射 ```map```
 ```#include <map>```

 提供 **对数** 时间的有序键值对结构，底层原理是 **红黑树**

 ```c++
 map<string , int> a;
 a["apple"] = 1;
 a["banana"] = 2;
 a["orange"] = 3;
 ```

 |性质|解释|map|multimap|unordered_map|
 |:-:|:-:|:-:|:-:|:-:|
 |互异性|键不能重复|✔️|❌(任意次)|✔️|
 |无序性|键没有顺序|❌(从小到大)|❌(从小到大)|✔️|

  * ## 6.1 构造
    ```c++
    map<键类型 , 值类型 , 比较器> 变量名;
    ```
    比较器默认为 ```less<键类型>```

  * ## 6.2 基本用法
    |作用|用法|示例|
    |:-:|:-:|:-|
    |增改查|```变量[](键) = 值```|```a["apple"] = 1```|
    |删除|```.erase(键)```|```a.erase("apple")  //删除该键```|
    |查找|```.find(键)```|```a.find("apple")    //返回键对应的迭代器```|
    |查找个数|```.count(键)```|```a.count("apple") // map里为0或1```|
    |判空，清空，取长度|```.empty()``` ```.clear()``` ```.size()```|同 ```vector```|

    **增删改查**时间复杂度 $O(\log{n})$

  * ## 6.3 遍历
    * 使用迭代器
      ```c++
      for( map<string, int>::iterator it = a.begin(); it != a.end(); ++it)
        cout << it->first << " : " << it->second << endl; //迭代器it是一个指针
      ```

    * 基于范围的循环（C++11）
      ```C++
      for(auto &变量名 : map变量名)
        cout << 变量名.first << " : " << 变量名.second << endl;   //此处变量是map的一个元素，first是键，second是值
      ```
  * ## 6.4 注意事项

    **1.使用中括号访问时会有默认值**

    访问 ```a[键]``` 时，若键不存在，会自动插入并赋值为0
    
     * 此时 ```a.count(键)``` 会返回1。
    ```c++
    a["pear"]; // 如果"pear"键不存在，会自动插入并赋值为0
    ```

    **2. 不能使用迭代器计算下标差值**

    同set，不能使用迭代器计算下标

 # 9. 关于迭代器

  对于一个`vector`，可以使用下标遍历
  ```cpp
  for (int i = 0; i < vec.size(); i++){
      cout << vec[i] << endl;
  }
  ```

  或者使用迭代器
  ```cpp
  for (vector<int>::iterator it = a.begin() ; it != a.end() ; ++it){
      cout << it* << endl;
  }
  ```

 * `a.begin()` 即为指向容器的第一个元素

 * `a.end()` 即为指向容器的最后一个元素的 **下一个位置**

 * 迭代器与指针类似，可使用 `*` 进行解引用操作

  ## 1. 为什么要使用迭代器？

  很多数据结构并不是线性，对于非线性数据结构，无法使用下标进行遍历。

  迭代器可以定义某个数据结构的遍历方式，通过迭代器的增减遍历非线性结构。

  例如，set的实现是基于红黑树的，而红黑树是一种自平衡的二叉搜索树。我们可以通过迭代器来遍历set中的元素。

  ```c++
  for(set<int>::iterator it = st.begin() ; it != st.end() ; ++it)
  cout<< *it << endl;
  ```

  ## 2. 迭代器用法

  在 `vector` 中具有较完整的用法

  * `.begin()` 头迭代器

  * `.end()` 尾迭代器

  * `.rbegin()` 反向头迭代器

  * `.rend()` 反向尾迭代器

  * `.prev(迭代器)` 返回前一个迭代器

  * `.next(迭代器)` 返回后一个迭代器

  其他容器中不一定具有全部功能，比如 `set` 无法相减求距离。

  ## 3. 常见问题

  * `.end()`  和 `.rend()` 指向的位置是无意义的值（最后一个元素的**后一个元素**） 

  * **注意不同容器支持的迭代器运算不同**

  * **注意删除操作时迭代器的失效问题**（比如对最后一位删除时会导致越界超时）
